# 蚯蚓投喂 SOP (标准操作流程)

## 基本参数
| 参数 | 推荐值 | 备注 |
|------|--------|------|
| 投放密度 | 1-1.5 万条/m² | 视品种而定(大平二号等) |
| 日投喂量 | 体重的 50-100% | 夏季取高值，冬季取低值 |
| 饲料含水率 | 60-70% | 手握成团、松手即散为宜 |
| 适宜温度 | 15-25°C | <5°C 或 >35°C 停止投喂 |
| 适宜 pH | 6.5-7.5 | 用石灰或醋调节 |

## 投喂流程

### Step 1: 饲料预处理
1. 收集稻草/秸秆 → 粉碎至 2-5 cm
2. 加水堆沤发酵 7-15 天（温度需降至 30°C 以下方可使用）
3. 可混入牛粪(3:7 秸秆:牛粪)加速发酵

### Step 2: 投喂操作
1. 将发酵好的饲料均匀铺在养殖床表面
2. 厚度 5-8 cm，不可过厚（避免厌氧发热）
3. 投喂后轻洒水保湿

### Step 3: 日常检查
- 每日检查表面湿度，干燥时喷水
- 每 3-5 天翻动一次，观察蚯蚓活动状态
- 饲料消耗 70% 以上时补充新料

## 与水稻循环的衔接
```
稻草/秸秆 → 发酵 → 蚯蚓饲料
                        ↓
                    蚯蚓消化
                        ↓
                    蚯蚓粪肥 → 水稻底肥
```
