# 生物肥加工流程 (Processing)

## 三阶段路径

### 第一阶段：自用（初期）

蚯蚓粪简单堆沤 → 直接施到稻田。
- 无需复杂加工
- 不用考虑销售端
- 重点关注重金属是否达标（尤其淤泥来源）

### 第二阶段：标准化（中期）

过筛 + 基础检测 → 满足自用品控。
- 建立基本的质量管控流程
- 积累检测数据，为商品化做准备

### 第三阶段：商品化（远期）

过筛 + 全项检测 + 配方混合 + 包装 → 高端定制生物肥。
- 锁定高价值作物客群（中草药等）
- 按下游作物需求定制 NPK 配比

> 只有足够的溢价空间，才能支撑过筛、检测和包装的成本。所以商品化阶段必须锁定高价值作物客群。
