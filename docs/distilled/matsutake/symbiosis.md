# 松茸共生关系

## 与宿主树种的共生
松茸（Tricholoma matsutake）为典型外生菌根菌，需与活体树木形成共生关系。

### 常见宿主
- 赤松（Pinus densiflora）— 最常见
- 红松、黑松、云南松
- 部分栎属树种

### 养分交换机制
```
树木 ---(碳水化合物/糖类)--→ 松茸菌丝
松茸 ---(水分 + 矿质元素)--→ 树木根系
```

## 在循环链条中的角色
```
水稻秸秆 → 基质原料 → 松茸栽培 → 菌糠废料 → 蚯蚓饲料
```

### 菌糠产出估算
- 每 100kg 基质 → 约 60-70kg 菌糠（废料）
- 菌糠 C/N 比约 25-30:1，适合蚯蚓消化
