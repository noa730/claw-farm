# 松茸产量影响因素

## 关键变量
| 因素 | 影响方向 | 权重 |
|------|---------|------|
| 温差刺激（昼夜温差 ≥8°C） | 促进原基分化 | 高 |
| 空气湿度 85-95% | 保障子实体膨大 | 高 |
| 基质 C/N 比 | 影响菌丝活力 | 中 |
| 光照（散射光） | 引导子实体方向 | 低 |
| CO₂ 浓度 <1000ppm | 防止畸形菇 | 中 |

## 产量基准（参考值）
- 林下仿野生栽培：0.5-2 kg/m²·季
- 设施化栽培（试验阶段）：数据待补充

## 产量预测模型思路
```
yield = f(temp_curve, humidity_curve, substrate_quality, symbiosis_index)
```
- `temp_curve`: 30天温度曲线
- `humidity_curve`: 对应湿度曲线
- `substrate_quality`: 基质养分评分
- `symbiosis_index`: 共生关系健康度
