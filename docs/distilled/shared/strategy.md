# 商业逻辑 (Strategy)

## 核心商业原则

1. **轻资产启动** — 重资产投入（如光伏）通过引入合作方解决，不用自有资金硬投
2. **内部消化优先** — 各环节产出先满足链内需求，外部销售是锦上添花
3. **溢价来源于体系** — 产品溢价不靠品类本身，而是循环生态赋予的品质背书（零化肥、可溯源、生态闭环）
4. **多元收入对冲风险** — 通过一物多用、多环节产出分散单一农产品的价格波动和气候风险

## 销售渠道

- **稻米**：品牌直营 / 社区团购 / 礼品定制
- **松茸**：高端餐厅直供 / 精品电商
- **生物肥**：初期自用，远期对接高价值作物种植方（中草药等）
- **蚯蚓活体**：周边鱼塘养殖户 / 渔具店

## 核心愿景

1. **零废弃**：全链条物质闭环，无外排废料
2. **可溯源**：从基质到餐桌全程记录，支持扫码溯源
3. **可复制**：模式标准化后可向其他产区推广
