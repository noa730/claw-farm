# 项目时空背景 (Context)

## 项目来源与范围

本项目脱胎于 **2022年度梁子湖区梁子镇全域国土综合整治项目**（梁子湖区人民政府，2023年3月），总投资 19429.71 万元。

- **覆盖范围**：刘斌村、沙湾村、长岭居委会
- **总面积**：约 1065 公顷
- **硬约束**：地块划分来自全域整治规划，不可随意改动

## 区位信息
- **地点**：湖北省鄂州市梁子湖区梁子镇
- **经纬度**：30.39°N, 114.89°E
- **海拔**：11.7m（梁子湖最低）— 485.8m（四峰山最高），城区约 20-50m
- **地形**：长江南岸滨湖丘陵区。东南部为幕阜山余脉，中部为湖区平原（梁子湖、鸭儿湖等），西北部为长江冲积平原

### 季节节律
- 雨季：4月-9月（梅雨集中在 6-7 月）
- 旱季：10月-3月（相对少雨）
- 松茸窗口期：9月-11月 / 3月-5月（气温 15-20°C 适宜出菇）
- 水稻种植期：4月育秧，5月插秧，9-10月收割（单季稻）；部分地区可种双季稻

### 关键工程约束
- **旱改水**：1840.21 亩
- **占补平衡**：462.83 亩
- **高标准农田基础设施**：田间道 10609m、农渠 16421m、泵房 17 座
- **骨干道路体系**："三横五纵"，总长 17434m，两侧种本地果树形成生态廊道

## 水系与灌溉
- **水源类型**：梁子湖水系（湖北第二大淡水湖）、长江支流、长港水系
- **水质**：梁子湖水质优良（Ⅱ-Ⅲ类），保安湖常年Ⅱ类水质，适合灌溉
- **灌溉设施**：配套灌溉坑塘 12 口、PE管（DN250mm）6781m、截流沟 7844m

## 周边生态
- **林木**：杉树、松树、杨树、槐树；果树有桃、枣、橘等
- **与菌菇共生的潜力宿主**：松树已确认存在于林地中，栎树分布需实地调查
- **现有生物多样性**：梁子湖区域生态良好，水草丰富，已有生态农业示范带（百里长港线）
- **铜铁海生态旅游园区**：位于沙湾村，规划面积约 2km²，水面约 400 亩
- **保安湖**：紧邻项目区，Ⅱ类水质，与梁子湖相连
