# 项目时空背景 (Context)

## 区位信息
- **地点**：湖北省鄂州市
- **经纬度**：30.39°N, 114.89°E
- **海拔**：11.7m（梁子湖最低）— 485.8m（四峰山最高），城区约 20-50m
- **地形**：长江南岸滨湖丘陵区。东南部为幕阜山余脉，中部为湖区平原（梁子湖、鸭儿湖等），西北部为长江冲积平原

### 季节节律
- 雨季：4月-9月（梅雨集中在 6-7 月）
- 旱季：10月-3月（相对少雨）
- 松茸窗口期：9月-11月 / 3月-5月（气温 15-20°C 适宜出菇）
- 水稻种植期：4月育秧，5月插秧，9-10月收割（单季稻）；部分地区可种双季稻
