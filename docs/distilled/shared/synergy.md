# 循环链路总图 (Synergy)

## 三层"一物多用"逻辑

| 层次 | 含义 | 示例 |
|------|------|------|
| 物质流循环 | A 的废料/产出 → B 的原料 | 稻秆→菌菇基质、菌糠→蚯蚓饲料、蚯蚓粪→水稻肥料 |
| 空间复合利用 | 同一地块叠加多种功能 | 林下种菌菇、林下养鸭、鱼塘上方光伏 |
| 产出多向输出 | 同一产物服务多个去向 | 蚯蚓→改土/消化淤泥/喂鱼/销售/生物肥原料 |

## 物质流闭环（扩展版）

在 CLAUDE.md 四环节基础上扩展，加入外部输入和多向输出：

```
外部输入：河道清淤淤泥(60万吨) ──→ 蚯蚓养殖
                                        │
水稻 ──稻秆──→ 菌菇 ──菌糠──→ 蚯蚓 ──蚯蚓粪──→ 有机肥 ──→ 水稻
                 │                  │                  │
                 ├→ 鲜品销售        ├→ 活体销售         ├→ 自用（稻田）
                 └→ 干品加工        ├→ 喂鱼             └→ 高端定制肥
                                   └→ 土壤改良               （中草药等）
```

### 物质流接口规格

| 上游 | 产出物 | 下游 | 接口规格 |
|------|--------|------|---------|
| 水稻 | 稻秆 | 菌菇 | 基质原料，需粉碎至 1-3cm |
| 菌菇 | 菌糠废料 | 蚯蚓 | C/N 比 25-30:1，含水量 60-70% |
| 蚯蚓 | 蚯蚓粪 | 有机肥加工 | 需过筛、检测重金属达标 |
| 有机肥 | 成品肥 | 水稻 | 施用量依据土壤检测结果 |
| 河道清淤 | 淤泥 | 蚯蚓 | 保护区干净淤泥，需重金属检测 |

## 空间复合利用矩阵

| 基础空间 | 叠加功能 | 阶段 |
|---------|---------|------|
| 林地 | 菌菇种植（林下） | 近期 |
| 林地 | 林下养鸭/散养鸡 | 中期 |
| 林地 | 林药套种 | 中期 |
| 鱼塘水面 | 光伏板（合作方投入） | 远期 |
| 稻田 | 稻田画/农旅体验 | 中期 |
| 生态廊道（果树） | 采摘观光 | 中期 |

## 开放性原则

> 以上仅为已识别的循环路径。所有可以形成循环的新路径都应被纳入，包括当前规划未覆盖的部分。本文档持续演进。
