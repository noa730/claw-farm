# 水稻生长阶段说明

## 生长周期总览

| 阶段 | 时间(天) | 关键特征 | 管理要点 |
|------|----------|----------|----------|
| 秧苗期 | 0-25 | 种子萌发→3叶期 | 控温保湿，适量施种肥 |
| 分蘖期 | 25-55 | 腋芽萌发，茎数增加 | 浅水促蘖，重施分蘖肥 |
| 拔节期 | 55-75 | 茎秆伸长，节间拉长 | 适时晒田，控制无效分蘖 |
| 抽穗期 | 75-95 | 穗从叶鞘抽出，开花 | 保持水层，防治病虫害 |
| 灌浆期 | 95-115 | 籽粒充实，淀粉积累 | 干湿交替，叶面追肥 |
| 成熟期 | 115-135 | 籽粒变硬，穗下垂 | 排水落干，适时收割 |

## 根系发育深度参考
