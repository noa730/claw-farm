# skill-env-monitor

## 功能
读取环境传感器数据，与知识库中的最适生长条件对比，判断当前环境是否适宜。

## 输入
```json
{
  "temperature": 23.5,
  "humidity_air": 88,
  "humidity_substrate": 63,
  "co2_ppm": 800,
  "light_lux": 500
}
```

## 执行逻辑
1. 加载 `docs/distilled/matsutake/growth_conditions.md` 中的阈值表
2. 逐项比对当前值与最适范围
3. 生成状态报告：每项标注 ✅ 适宜 / ⚠️ 偏离 / ❌ 超限

## 输出
```
【环境诊断】
  温度: 23.5°C → ✅ 菌丝期适宜 (22-25°C)
  空气湿度: 88% → ✅ 出菇期适宜 (85-95%)
  基质含水量: 63% → ✅ 正常 (60-65%)
  CO₂: 800ppm → ✅ 正常 (<1000ppm)
  光照: 500lux → ✅ 散射光充足

【综合评估】当前环境适合菌丝生长阶段
【预警】无
```

## 预警规则
- 温度偏离最适 ±3°C → ⚠️ 预警
- 温度偏离最适 ±5°C → ❌ 告警，建议立即调控
- 湿度 <80% 且处于出菇期 → ❌ 告警
- CO₂ >1500ppm → ❌ 通风不足
