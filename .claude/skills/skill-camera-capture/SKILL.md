# 摄像头采集技能 (Camera Capture)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 拍照、拍一张、看看现场、摄像头
- 图片、照片、视觉检查、长势观察

## 适用范围
全局技能，为所有 Agent 提供视觉输入：
- matsutake-expert → 观察菌丝覆盖、子实体形态、污染斑点
- rice-expert → 观察叶色、分蘖数、病虫害症状
- worm-expert → 观察蚯蚓活跃度、饲料分解状态

## 执行流程
1. **调用摄像头**：
   - 若有本地摄像头 → 调用系统摄像头拍照
   - 若有远程摄像头（IP Camera）→ 通过 URL 抓取画面
   - 若无设备 → 提示用户手动上传照片
2. **保存照片**：存入 `assets/images/captures/YYYY-MM-DD_HHmmss.jpg`
3. **返回路径**：将照片路径返回给调用方 Agent
4. **由 Agent 分析**：本技能只负责采集，视觉分析由各专家 Agent 自行完成

## 拍摄参数建议
| 场景 | 建议距离 | 备注 |
|------|---------|------|
| 松茸子实体 | 10-30cm | 微距，关注形态和色泽 |
| 菌丝覆盖 | 20-50cm | 需均匀光照，避免反光 |
| 水稻田间 | 1-2m | 拍摄整株，含叶片细节 |
| 蚯蚓床 | 30-50cm | 翻开表层后立即拍摄 |

## 输出格式
```
【拍摄完成】
  文件: assets/images/captures/2026-04-01_153022.jpg
  设备: 本地摄像头 / IP Camera / 用户上传
  分辨率: 1920×1080
  时间: 2026-04-01 15:30:22

→ 已交由 {agent_name} 进行视觉分析
```

## 注意事项
- 照片仅存储在本地 `assets/images/captures/`，不自动上传
- 建议 `.gitignore` 中排除 `assets/images/captures/` 避免仓库膨胀
