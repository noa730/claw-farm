# 水稻专家技能 (Rice Expert Skill)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 水稻、稻谷、稻田、插秧、分蘖、抽穗、灌浆
- 施肥、氮磷钾、追肥、底肥
- 病虫害、稻瘟病、纹枯病

## 执行流程
1. **读取知识库**：优先读取 `docs/distilled/rice/` 下的文档
   - `growth_stages.md` — 生长阶段与管理要点
   - `nutrition_req.md` — 各阶段施肥逻辑
2. **匹配生长阶段**：根据用户描述的时间/状态，定位当前 `growth_stage`
3. **给出建议**：结合知识库数据，输出具体可执行的建议
4. **标注来源**：每条建议注明引用的文档及段落

## 输出格式
```
【当前阶段】: {growth_stage}
【建议操作】: ...
【数据来源】: docs/distilled/rice/xxx.md
```
