# 孢子分析技能 (Spore Analysis)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 孢子、孢子活性、孢子萌发、孢子密度
- 菌种质量、接种、活力检测

## 执行流程
1. **获取数据**：读取孢子检测数据（显微镜计数 / 染色活性率）
2. **加载知识库**：读取 `docs/distilled/matsutake/growth_conditions.md`
3. **分析指标**：
   - 孢子密度（个/mL）
   - 活性率（台盼蓝染色排斥法）
   - 萌发率（培养 48h 后统计）
   - 污染率（杂菌孢子占比）
4. **质量判定**：对照阈值给出菌种质量等级
5. **标注来源**：引用知识库或检测标准

## 判定标准
| 指标 | 优 | 良 | 不合格 |
|------|----|----|--------|
| 活性率 | ≥90% | 70-89% | <70% |
| 萌发率 (48h) | ≥80% | 60-79% | <60% |
| 污染率 | <1% | 1-5% | >5% |
| 密度 (个/mL) | ≥1×10⁶ | 5×10⁵-1×10⁶ | <5×10⁵ |

## 输出格式
```
【孢子分析报告】
  活性率: 92% → 优
  萌发率: 85% → 优
  污染率: 0.3% → 优
  密度: 1.2×10⁶/mL → 优

【综合评级】A级（可直接用于接种）
【建议】当前批次菌种质量优良，建议 72h 内完成接种
【数据来源】docs/distilled/matsutake/growth_conditions.md
```
