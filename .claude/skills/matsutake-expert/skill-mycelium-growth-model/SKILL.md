# 菌丝生长模型技能 (Mycelium Growth Model)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 菌丝、菌丝生长、蔓延速度、基质配比
- 菌丝培养、满袋时间、走菌

## 执行流程
1. **获取基质参数**：配方比例、含水量、C/N 比
2. **获取环境参数**：温度、湿度（可调用全局 skill-env-monitor）
3. **加载知识库**：读取 `docs/distilled/matsutake/growth_conditions.md` + `yield_factors.md`
4. **运行模型**：
   - 基础速率：温度 → 菌丝日伸长量（mm/d）
   - 基质修正：C/N 比偏离最适值 → 速率折减
   - 含水量修正：偏离 60-65% → 速率折减
   - 预测满袋/满床时间
5. **标注来源**

## 基础生长速率参考
| 温度 (°C) | 日伸长量 (mm/d) | 状态 |
|-----------|----------------|------|
| 15 | 1.5-2.0 | 缓慢 |
| 20 | 3.0-4.0 | 正常 |
| 23 | 4.5-5.5 | 最适 |
| 25 | 4.0-5.0 | 正常 |
| 28 | 2.0-3.0 | 受抑 |
| >30 | <1.0 | 高温抑制 |

## 修正系数
- **C/N 比修正**：最适 25-30:1 → 系数 1.0；每偏离 5 → ×0.85
- **含水量修正**：最适 60-65% → 系数 1.0；每偏离 5% → ×0.8
- **pH 修正**：最适 4.5-5.5 → 系数 1.0；每偏离 0.5 → ×0.9

## 输出格式
```
【菌丝生长预测】
  基质: 稻秆粉50% + 松针腐殖质25% + 米糠15% + 石膏2%
  C/N 比: 28:1 → 修正系数 1.0
  含水量: 63% → 修正系数 1.0
  温度: 23°C → 基础速率 5.0 mm/d

  预测日伸长: 5.0 mm/d
  培养袋规格: 15×30cm
  预测满袋时间: 约 28-35 天

【风险提示】温度波动 >±3°C 会显著降低菌丝活力
【数据来源】docs/distilled/matsutake/growth_conditions.md
```
