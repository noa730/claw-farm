# 共生匹配技能 (Symbiotic Matching)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 共生、菌根、宿主树、养分交换
- 赤松、黑松、红松、栎树、松茸共生匹配

## 执行流程
1. **确认宿主信息**：树种、树龄、根系健康度
2. **加载知识库**：读取 `docs/distilled/matsutake/symbiosis.md`
3. **计算匹配度**：
   - 树种适配性（松茸对不同树种的偏好指数）
   - 根系活力评分（树龄 → 根系分泌物丰度）
   - 土壤条件适配（pH、有机质是否支持菌根形成）
4. **估算养分交换效率**：
   - 碳水化合物供给率：树木 → 菌丝
   - 矿质元素回馈率：菌丝 → 树木（磷、钾、微量元素）
5. **标注来源**

## 树种适配指数
| 树种 | 适配指数 | 备注 |
|------|---------|------|
| 赤松 (P. densiflora) | ★★★★★ | 天然首选宿主 |
| 红松 (P. koraiensis) | ★★★★ | 东北产区常见 |
| 黑松 (P. thunbergii) | ★★★ | 沿海耐盐碱 |
| 云南松 (P. yunnanensis) | ★★★★ | 西南产区主力 |
| 栓皮栎 (Q. variabilis) | ★★ | 可形成菌根但效率低 |

## 输出格式
```
【共生匹配分析】
  宿主: 赤松 (20年生)
  适配指数: ★★★★★
  根系活力: 良好（壮年期，分泌物丰富）
  土壤 pH 5.5: ✅ 适合菌根形成

【养分交换效率】
  碳供给率: ~15 mg C/g菌丝·日（估算）
  磷回馈率: ~0.8 mg P/棵树·日（估算）

【匹配结论】高度适配，建议在该树周围 1-3m 范围内布设接种点
【数据来源】docs/distilled/matsutake/symbiosis.md
```
