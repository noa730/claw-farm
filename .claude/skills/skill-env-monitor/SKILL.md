# 环境监测技能 (Environment Monitor)

## 触发条件
当用户提问涉及以下关键词时激活本技能：
- 温度、湿度、光照、CO₂、土壤湿度
- 环境数据、传感器、现场条件、气象
- 当前环境、适不适合、能不能种

## 适用范围
全局技能，可被任何 Agent 调用：
- matsutake-expert → 判断菌丝/出菇环境
- rice-expert → 判断水稻田间条件
- worm-expert → 判断蚯蚓养殖环境

## 执行流程
1. **数据获取**（按优先级）：
   - 优先：读取传感器实时数据（若已接入 IoT 数据源）
   - 次选：用户手动输入当前数据
   - 兜底：根据项目所在地区 + 当前日期，推断季节性气候范围
2. **整理输出**：统一格式返回环境数据
3. **交由调用方判断**：本技能只负责采集/推断数据，不做农业建议

## 兜底推断逻辑
当无传感器数据也无用户输入时：
- 询问用户项目所在省份/地区
- 根据地区 + 月份，给出气候统计范围（参考中国气象局历史数据）
- 明确标注"推断值，非实测"

## 输出格式
```
【环境数据】
  数据来源: 实测 / 用户输入 / 推断
  采集时间: YYYY-MM-DD HH:mm

  气温: 23.5°C
  空气湿度: 85%
  土壤温度: 20°C
  土壤湿度: 62%
  光照: 500 lux
  CO₂: 800 ppm
  风速: 1.2 m/s

【备注】若为推断值，标注置信区间
```
